# dotfiles
 
## Debian (Gnome)

```shell
sudo apt install git
git clone https://github.com/adamgian/dotfiles ~/.dotfiles
```

#### Setup apps and tools
```shell
sh ~/.dotfiles/os/debian/setup.sh
```

#### Configure system
```shell
sh ~/.dotfiles/os/debian/configure.sh
```

#### Load in dotfiles
```shell
sh ~/.dotfiles/os/debian/dotfiles.sh
```
